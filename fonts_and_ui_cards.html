Fonts and UI Cards

For this assignment the task has been about how to implement fonts, a hierarchy to the fonts, and how to design the UI cards. We had some golden (strict) rules to follow.

  • Only 1-2 fonts!
  • No display fonts on the buttons!
  • Everything should be in rem! (for responsiveness)
  • - and the most important one --> No borders on the UI cards!
Another criteria have been to use CSS flex for the UI cards and how to style the call to action (CTAs) so the user understand the purpose of the buttons on the cards just from looking at them.
