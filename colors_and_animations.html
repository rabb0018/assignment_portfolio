Colors and Animation

For this assignment the task is about how to choose a color palette for a landing page design, make a dark theme and to add animations that have been taught in the lectures.

The most important tools for this assignment has been the color tester and the 60-30-10 rule. It has been a huge help when trying to make a color palette, which is actually really time consuming and hard to do, and for how much each color should fill on the website.

I've tried to implement the same colors on both the light and dark theme, with small adjustments to the hue and saturation, so the colors don't clash together and make the text unreadable.
